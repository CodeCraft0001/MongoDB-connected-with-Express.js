# Express + MongoDB CRUD (Learning Project)

This is a beginner-level backend project built to understand:
- MongoDB connection
- Mongoose schemas and models
- CRUD operations via Express APIs
- Difference between mongosh and application-level CRUD

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## How to Run
1. Install dependencies  
   `npm install`

2. Add environment variables  
   Create `.env` using `.env.example`

3. Start server  
   `node server.js`

4. Test APIs using Postman
